Web + Mobile System Design

DIGF-6039-001

2018 Spring/Summer
Students study digital architecture and implementation strategies that effectively create cutting-edge mobile and cross-platform solutions, addressing both front end (HTML, CSS, Javascript W3C APIs, jQuery), and server-side (PHP, MySQL, Ajax, security) design.

The course provides an overview of the web and mobile landscape and a technical immersion into programing and scripting options, tools, systems, processes, hardware, web apps and apps. Students integrate digital components into open-source solutions through project assignments. Students are guided on the utilisation of API's for services (Google Maps and Twitter), sensors and environments that enable augmented reality, image and speech recognition.
